# < 편의점 - 꿀팁 레시피 >

## 카테고리 분류

- 카테고리 분류 기준 : 최종 음식 종류
- 각 레시피 당 카테고리는 최대 2개
- ex) 면, 국/찌개, 안주, 리조또, 음료, 떡볶이, (필요하면 더 추가)

## 추가 설명

- 체크박스 혹은 콤보박스로 카테고리 하나만 선택
- 해당 카테고리에 맞는 레시피 리스트들 보여줌
- 레시피 하나 선택 시 세부 내용 (재료, 레시피) 보여줌
- 이전 화면으로 돌아가는 버튼 필요
