## Project Plan

### 아이디어

저희는 대학생을 위한 메뉴(레시피)를 추천해주는 프로그램을 제작합니다.

### 프로그램의 추상화

프로그램은 사용자가 화면에 제시된 항목에서 본인이 원하는 옵션을 체크합니다. \
그리고 선택된 옵션들을 바탕으로 프로그램이 필터링을 거쳐주고 사용자에게 메뉴를 추천해줍니다.
